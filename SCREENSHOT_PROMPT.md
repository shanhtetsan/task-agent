# Task Agent Screenshot Prompt

Use this prompt with any screenshot/mockup image generator when you need a polished placeholder for `public/screenshot.png`.

## Prompt

```text
Create a realistic desktop app screenshot for a student productivity tool called "Task Agent".

Layout requirements:
- Left side: "Today" dashboard with a focus card and upcoming tasks.
- Right side: AI agent chat panel open, showing a multi-turn conversation.
- Include one preview card generated by the agent for a task ("Coursework", "Event", or "Personal").
- Show a small monthly calendar widget with task dots/indicators.
- Keep a floating action button in the bottom-right corner.

Style:
- Clean React/Tailwind style UI
- Accessible contrast, readable typography
- Light theme
- Modern but minimal
- Academic/student context

Content constraints:
- Use realistic but fake data only.
- No personal identifiers, no real email addresses, no phone numbers.
- No brand logos beyond neutral interface elements.

Output:
- 16:9 image
- PNG format
- Save as public/screenshot.png
```

## Quick Capture Alternative (Real App)

If you prefer a real product screenshot instead of generated art:

1. Run `npm run dev`
2. Open the app in Chrome at `http://localhost:5173`
3. Set window size to 1440x900 or 1920x1080
4. Open the agent panel and keep Today dashboard visible
5. Capture and save as `public/screenshot.png`
